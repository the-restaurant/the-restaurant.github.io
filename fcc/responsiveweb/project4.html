Introduction

FoobyScript is a programming language created by Meep Fooby. FoobyScript is an offshoot of JavaScript and TypeScript. Its syntax is extremely similar to both and only has minor differences. The biggest difference between FoobyScript and its contemporaries is that FoobyScript is not meant to run in the web browser, or any other environment that runs JavaScript, either as compiled to machine code or interpreted.

The advantage of this is that FoobyScript can be optimized to run on its target hardware, the Sony CrabStation.
